# TodoList

A simple SwiftUI-based todo list app.

## Features
- search todos
- edit description

## Demo Video

[▶️Click here to watch the screen recording](<https://drive.google.com/file/d/1Z0ZkOm26QyQXU24zfXLiDVBBiUEwUGRy/view?usp=sharing>)

## Requirements
- Xcode 
- iOS 14+

## Running the App
1. Open `todolist.xcodeproj` in Xcode.
2. Select an iOS Simulator or a connected device.
3. Click **Run** to build and launch the app.
